.XLSX naar .TXT Convertor voor Orange Cloud Telephony

Waarvoor dient deze Tool?

je kan in deze tool een .XLSX bestand makkelijk omvormen naar een .TXT bestand.
Met als bedoeling dat je makkelijk je telefoon contacten in bulk kan toevoegen aan je Cloud telephony systeem.
Voor je deze tool gebruikt raden we je eerst aan om de handleiding eerst eens door te nemen.

Instructies

• Voeg een (.XLSX) bestand toe door te klikken op de "Kies bestand" knop.
• Klik op de "Download converted bestand" knop om je omgevormde (.TXT) bestand te downloaden.
